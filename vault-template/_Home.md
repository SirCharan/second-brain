# 🏠 Home — Second Brain

Central map of the vault. Notes live in project folders; hubs are `_MOC-*`.

> [!info] Getting started
> - Talk to Claude Code normally — every session is auto-captured into `Daily/`.
> - Curate a durable note anytime: `/second-brain capture "<fact>"`.
> - Search when stuck: `/second-brain pull "<terms>"`.
> - Audit health: `/second-brain health`.
