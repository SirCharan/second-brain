# Welcome — how a note is shaped

🟢 **active**

This is what a curated note looks like. Delete this `example/` folder once you've read it.

> [!tip] The shape
> Frontmatter v2 (above) + `# H1 title` + an emoji status chip + colored callouts wrapping
> the key facts + a `## Related` section linking the folder hub and 2–4 related notes.

> [!info] Status chips
> 🟢 **active** · 🟡 **watch** · ⚫ **retired** · add 🔴 **real-money** for live-order/spend/key notes.

## Details
One concept per note. Link liberally with `[[wikilinks]]`. Newest fact wins — never delete;
retire with `status: retired` + a `supersedes:` pointer to the old note.
